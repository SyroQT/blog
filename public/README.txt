Required Icon Files
==================

Please create the following files in this directory:

1. favicon.ico
   - Multiple sizes: 16x16, 32x32, and 48x48 pixels
   - Used for browser tabs and bookmarks

2. apple-icon.png
   - Size: 180x180 pixels
   - Used for iOS home screen icons

3. android-chrome-192x192.png
   - Size: 192x192 pixels
   - Used for Android home screen icons

4. android-chrome-512x512.png
   - Size: 512x512 pixels
   - Used for Android home screen icons

5. og-image.jpg
   - Size: 1200x630 pixels
   - Used for social media sharing previews

You can create these files using:
1. Online tools:
   - https://realfavicongenerator.net/ (for favicon and app icons)
   - https://www.opengraph.xyz/ (for OpenGraph images)

2. Image editing software:
   - Adobe Photoshop
   - GIMP
   - Figma

Note: Make sure all images are optimized for web use to ensure fast loading times.
